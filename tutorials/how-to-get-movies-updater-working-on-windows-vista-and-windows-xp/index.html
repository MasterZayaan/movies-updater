--- layout: default --- How to install Movies Updater on Windows Vista/XP How to install Movies Updater on Windows Vista/XP

So, you want to install Movies Updater on Windows Vista or Windows XP.

Well, while we don't recommend it, you can install the app on Vista or XP but you'll need to do a lot more then just run an installer.

Installing Movies Updater on Windows Vista:

First up, you want to download the app and copy then copy the app to wherever you want then run it.

Once you have run the app, install it!

Now, what you will need to do is open Command Prompt and then cd to C:\Program Files (x86)\Movies Updater or C:\Program Files\Movies Updater and then type in Starter or Starter.bat.

Issues:

Windows Vista doesn't think that the app is a valid Win32 app due to it meant to run on newer OSs.

Installing Movies Updater on Windows XP:

Yet again, download the app and copy it to your desktop for easy access.

This time, you'll need to download 7-Zip and install it.

Once you have downloaded and installed 7-Zip, right-click on the installer and click Open Archive.

Then select all the files and folders in the list and press Extract.

Now if you have a x86/32 bit system, change the extracting path to C:\Program Files\Movies Updater.

But if you have a x64/64 bit system, change the extracting path to C:\Program Files (x86)\Movies Updater.

DON'T CHANGE ANYTHING ELSE, OTHERWISE MOVIES UPDATER WON'T INSTALL PROPERLY!

Once extracted do the same process that we did for Vista, open Command Prompt, go to C:\Program Files (x86)\Movies Updater for x64/64 bit systems or go to C:\Program Files\Movies Updater for x86/32 bit systems.

Finally, once you have gone through all those steps, type in Starter or Starter.bat and enjoy Movies Updater.

I have also provided all the tools you need to do this, as well as a batch script to put on your desktop so that you can open Movies Updater with ease.

Issues:

XP has the same issue with Vista as the EXE is not a valid Win32 app due to it being designed for newer operating systems.

But XP can't launch the Movies Updater installer for some weird reason.

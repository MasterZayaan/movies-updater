--- layout: default --- How to install Movies Updater on Windows Vista/XP How to install Movies Updater on Windows Vista/XP

So, you want to install Movies Updater on Windows Vista or Windows XP.

Well, while we don't recommend it, you can install the app on Vista or XP but you'll need to do a lot more then just run an installer.

Windows Vista:

First up, you want to download the ZIP file and copy the latest version to wherever you want then run it.

Once you have run the app, install it!

Now, what you will need to do is open Command Prompt and then cd to C:\Program Files (x86)\Movies Updater or C:\Program Files\Movies Updater and then type in Starter or Starter.bat.

Problem:

Windows Vista doesn't think that the app is a valid Win32 app due to it meant to run on newer OSs.
